# Product Context

register: product

## Product Purpose

Review App is a desktop PR review tool for developers. It turns GitHub pull request diffs into AI-assisted narrative walkthroughs so reviewers can understand intent, risk, and implementation details faster.

## Users

- Software engineers reviewing code across repositories
- Maintainers triaging incoming pull requests
- AI-assisted development teams who need clear PR context

## Tone

Professional, focused, trustworthy, and efficient. The app should feel like a polished developer tool: precise, calm, dense when useful, and never decorative for its own sake.

## Design Principles

- Prioritize scanability for long diffs and repository lists
- Use restrained color with clear semantic states
- Make hierarchy obvious through spacing, type scale, and contrast
- Preserve developer-tool familiarity while avoiding bland defaults
